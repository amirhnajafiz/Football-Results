Hacking Computer Memory Details

Access to other parts of memory with C programming language. How does C programming language allows us to access every part of memory? How we can secure our system againts this problem? In this research we are going to talk about Buffer Overflow and it is a common cause of security vulnerabilities, as not checking the bounds of an array before indexing it can result in malicious modifications to your program at runtime.
